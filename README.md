# ANC Field Tech Checklist

Mobile-friendly Next.js app for ANC field techs.

## Run locally
1. Install Node.js
2. Open a terminal in this folder
3. Run:
   npm install
   npm run dev
4. Open http://localhost:3000

## Deploy on Vercel
1. Create a GitHub repository
2. Upload all files from this folder
3. Go to Vercel
4. Import the GitHub repository
5. Click Deploy
